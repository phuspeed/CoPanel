🚀 HOW TO USE

1️⃣ INSTALLATION (One Command)
   
   sudo bash scripts/install.sh
   
   This will:
   ✓ Install all system dependencies
   ✓ Set up Python virtual environment
   ✓ Install Python & Node packages
   ✓ Build frontend assets
   ✓ Configure Nginx reverse proxy
   ✓ Create Systemd service
   ✓ Start all services
   ✓ Verify everything works

2️⃣ ACCESS THE PANEL

   Web UI:        http://localhost:8686
   Backend API:   http://localhost:8000
   API Docs:      http://localhost:8000/docs

3️⃣ CREATE YOUR FIRST MODULE

   Backend:
   $ mkdir -p backend/modules/mymodule
   $ cat > backend/modules/mymodule/router.py << 'EOF'
   from fastapi import APIRouter
   router = APIRouter()
   
   @router.get("/hello")
   async def hello():
       return {"message": "Hello from mymodule!"}
   EOF
   $ touch backend/modules/mymodule/__init__.py

   Frontend:
   $ mkdir -p frontend/src/modules/mymodule
   $ cat > frontend/src/modules/mymodule/config.ts << 'EOF'
   import MyModule from './index';
   export default {
     name: 'My Module',
     path: '/mymodule',
     icon: 'Zap',
     component: MyModule,
   };
   EOF
   $ cat > frontend/src/modules/mymodule/index.tsx << 'EOF'
   export default function MyModule() {
     return <div className="p-8"><h1>My Module!</h1></div>;
   }
   EOF

   Deploy:
   $ systemctl restart lvp-panel

   Your module appears automatically! ✨

🎨 PLUGGABLE ARCHITECTURE EXPLAINED

The system is designed for ZERO configuration when adding modules:

Backend Module Discovery:
1. Any folder in backend/modules/
2. Must have router.py exporting 'router'
3. Loader automatically finds and registers it
4. Available at /api/{module_name}/endpoint
5. Restart service to load new modules

Frontend Module Discovery:
1. Any folder in frontend/src/modules/
2. Must have config.ts exporting default object
3. Registry automatically finds and registers it
4. Appears in sidebar + gets route
5. Rebuild frontend (npm run build) to load

Result: ADD A FOLDER, NOT CODE. THAT'S IT! 🚀

🛠️ DEVELOPMENT SETUP

Backend Development:
   cd backend
   source venv/bin/activate          # Or: venv\Scripts\activate on Windows
   pip install -r requirements.txt
   python -m uvicorn main:app --reload
   # Backend at http://localhost:8000 (auto-reloads on changes)

Frontend Development:
   cd frontend
   npm install
   npm run dev
   # Frontend at http://localhost:5173 (with hot reload)
   # API calls auto-proxy to http://localhost:8000
