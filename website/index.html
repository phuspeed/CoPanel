Trình quản lý VPS Linux thế hệ mới cực kỳ mạnh mẽ

Hệ thống quản lý VPS Pluggable mã nguồn mở đỉnh cao với giao diện tuyệt đẹp, hỗ trợ cài đặt các Module tùy chỉnh dễ dàng và mượt mà.

Bắt đầu ngay Xem tính năng

Hệ thống Module đa dạng & mạnh mẽ

Mọi tính năng quan trọng nhất cho máy chủ của bạn được đóng gói thành các Module có kiến trúc cắm (Pluggable Architecture).

System Monitor

Theo dõi thời gian thực các thông số phần cứng bao gồm CPU, RAM, Disk I/O, và quy trình xử lý mạng để tối ưu tài nguyên.

File Manager

Khám phá, chỉnh sửa, tải lên/tải xuống, sao chép, di chuyển, phân quyền file và nén ZIP trực tiếp trên trình duyệt.

Docker Manager

Tạo mới, dừng, chạy, xóa các Container và Images trực quan chỉ bằng một vài thao tác đơn giản.

Backup Manager

Tự động sao lưu dữ liệu máy chủ lên các dịch vụ đám mây nổi tiếng như Google Drive, OneDrive sử dụng Rclone mạnh mẽ.

SSL Certbot

Đăng ký và tự động gia hạn chứng chỉ SSL miễn phí từ Let's Encrypt cho mọi tên miền website của bạn.

Tường lửa & Bảo mật

Bật/tắt tường lửa UFW, cấu hình chặn và cho phép các cổng kết nối mạng an toàn để bảo vệ VPS toàn diện.

Kiến trúc mở rộng không giới hạn

Xây dựng các Module riêng của bạn bằng Python và React để tùy biến dịch vụ quản lý VPS theo cách riêng của mình.
