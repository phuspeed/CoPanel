Tính năng chính

Docker Manager

Quản lý container, image và compose trực tiếp trên giao diện web.

SSL tự động

Cấp và gia hạn chứng chỉ Let's Encrypt tự động cho các domain.

Backup & Restore

Sao lưu lên Google Drive / rclone, khôi phục nhanh khi cần.

File Manager

Duyệt, tải lên, chỉnh sửa file trực tiếp trong trình duyệt.

System Monitor

Theo dõi CPU, RAM, Disk và Network theo thời gian thực.

Pluggable Modules

Cài thêm module mở rộng không cần sửa core, dễ tích hợp.
